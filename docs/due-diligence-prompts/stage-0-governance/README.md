# Stage 0 — Diligence Governance, Epistemic Control, and Rigor Enforcement

*How do we prevent this diligence from lying to itself?*

**Purpose.** Establish the evidentiary constitution that governs all subsequent analysis. This is not a project plan — it is a system for preventing narrative drift, forcing confrontation with uncertainty, and protecting against confirmation bias.

**Produces.** A diligence constitution defining truth standards, evidence hierarchy, claim classification rules, hypothesis discipline, uncertainty preservation requirements, and red-team authority.

**Prerequisites.** None. This is the foundation.

**Failure condition.** If this stage does not make the diligence harder, it has failed.
